# LEDLNDChecker  *(Project cost $6)*
A really simple Green/Red LED feedback to tell you if your LND node is online

# Hardware needed: 
+ ESP32
+ Green LED
+ Red LED
+ x4 Jump Leads

# GPIO Map 

*GREEN LED*<br>
Pos = GPIO27*<br>
Neg = GND

*RED LED*<br>
Pos = GPIO26*<br>
Neg = GND
